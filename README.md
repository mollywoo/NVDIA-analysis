## üìñ Overview
This project is a web-based financial dashboard designed to provide a deep-dive analysis of **NVIDIA Corporation (NVDA)**. It moves beyond simple price tracking to offer a strategic assessment of the company's position in the AI hardware market.

The dashboard consolidates financial metrics, technical analysis, and a custom "Risk Radar" to evaluate the structural threats posed by competitors like Google and Meta.

## ‚ú® Key Features
* **Financial Deep Dive:** Visualizes Q3 FY2026 revenue trends, profit margins, and cash flow analysis.
* **Technical Analysis:** Real-time tracking of support/resistance levels, RSI indicators, and moving averages.
* **Risk Radar:** A unique interactive visualization assessing structural threats (e.g., Custom Silicon/TPUs, Geopolitics, Customer Concentration).
* **SWOT Analysis:** A comprehensive breakdown of Strengths, Weaknesses, Opportunities, and Threats.
* **Interactive Charts:** Built using `Chart.js` for responsive data visualization.

## üìä Data & Methodology
* **Financial Data:** Sourced from latest SEC Filings (10-Q/10-K) and Yahoo Finance.
* **Valuation Models:** DCF and PEG ratios are estimated based on current analyst consensus.
* **Risk Scoring:** The "Risk Radar" scores (0-10) are subjective metrics derived from current market sentiment and competitor analysis.
